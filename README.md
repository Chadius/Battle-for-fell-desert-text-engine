# Fell Desert Text Game Engine

This is a Text Based interface used to play the Battle for Fell Desert, a turn-based strategy game.

The game logic is located at https://github.com/Chadius/Battle-for-fell-desert-logic.git .
This is the new version of the previous project located at https://github.com/Chadius/Torrin-TBS .

# Installation
Clone this repository and run `npm install` to install dependencies.
Make sure the `logic` folder contains the Battle for Fell Desert logic submodule.

# Instructions
Type Q to quit the game.
